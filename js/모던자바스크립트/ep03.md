## 3.3 Node.js
>3.3.1 Node.js와 npm소개
>>브라우저에서만 동작하던 자바스크립트를 브라우저 이외의 환경에서 동작시킬수있는 실행환경이 node.js다

>3.3.2 Node.js설치
>>LTS는 long term support 장비적으로 안전된 버전임  
>>버전확인  

```
node -v  npm -v
``````

>3.3.3 Node.js REPL
>> cmd에서 node 치면 node환경으로 변경됨  
>> 종료 : ctrl +c 2번
